      - [ ] CSVファイルとしてまとめる

      - [ ] 所在地  
      - [ ] 売上規模（10〜300億の範囲）  
      - [ ] 担当部署（わかる場合）  
      - [ ] **メールアドレス**（最優先）  
      - [ ] 電話番号（任意）  
      - [ ] 問い合わせフォームURL（任意）  
      - [ ] 備考  
    - **納品物**  
      - [ ] CSVファイル
